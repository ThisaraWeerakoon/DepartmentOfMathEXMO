Heart Shape with Python Turtle

Welcome to this Python Turtle adventure! In this web page, we'll introduce you to a Python program that uses Turtle graphics to draw a heart shape. The heart shape is formed using mathematical equations and the fascinating properties of sin and cos functions.

The Heart Shape Equations

The heart shape is generated using the following mathematical equations:

Code --> python 

def hearta(k):
    return 15 * math.sin(k) ** 3

def heartb(k):
    return 12 * math.cos(k) - 5 * math.cos(2 * k) - 2 * math.cos(3 * k) - math.cos(4 * k)

Drawing the Heart Shape

To draw the heart shape, the Python program uses the Turtle graphics library. The heart shape is drawn by iterating through the equations for different values of k and plotting the corresponding (x, y) points on the canvas.

Code --> python 

from turtle import *
import math

speed(60000)
bgcolor("black")
for i in range(6000):
    goto(hearta(i) * 20, heartb(i) * 20)
    for j in range(5):
        color("#f73487")
    goto(0, 0)

done()

Running the Python Code

To see the heart shape drawing in action, save the Python code to a file with a .py extension, such as heart_shape.py. Open a terminal or command prompt, navigate to the folder containing the file, and execute the following command: 


python heart_shape.py

Behold the Heart Shape

As you run the Python program, a Turtle graphics window will pop up, and you'll witness the beautiful heart shape being drawn on the canvas. It's a mesmerizing display that showcases the artistry of mathematical functions and the power of Python Turtle!

Experiment and Explore

Feel free to experiment with different values in the hearta and heartb functions to create variations of the heart shape. You can change the colors, speed, or the size of the heart to customize your own unique creation!

Conclusion

With just a few lines of Python code and the magic of Turtle graphics, you've created a beautiful heart shape. It's a delightful way to explore the artistic side of Python while learning about the beauty of mathematics.
