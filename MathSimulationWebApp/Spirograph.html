Spirograph Art with Python Turtle

Are you fascinated by beautiful and intricate patterns? If so, you'll love what we're about to show you! In this blog post, we'll introduce you to a Python Turtle program that generates mesmerizing spirograph-like art. The best part? What is a Spirograph?

If you've never heard of a Spirograph before, it's a geometric drawing toy that produces elegant and elaborate circular patterns. By combining rotating gears and pens, users can create complex designs with ease. In this Python program, we'll recreate the magic of a Spirograph using the Turtle graphics library.

Code --> python 
from turtle import *
from colorsys import *

# Set up the drawing environment
bgcolor('black')
tracer(100)
pensize(4)
h = 0

# Function to draw the spirograph pattern
def draw(ang, n):
    circle(5 + n, 69)
    left(ang)
    circle(5 + 2 * n, 60)

# Position the turtle at the center
goto(0, 0)

# Generate the spirograph art
for i in range(500):
    c = hsv_to_rgb(h, 1, 1)
    h += 0.005
    color(c)

    up()
    draw(90, i)
    draw(180, i)
    down()

    draw(1 / 2, i - i)
    draw(180, i / 2)
    draw(120, i - i)

Running the Program

Save the code to a file with a .py extension, such as spirograph.py. Open a terminal or command prompt, navigate to the folder containing the file, and execute the following command:

Code --> python 
python spirograph.py

Watch the Magic Unfold

Once the program starts running, you'll witness the magic unfold before your eyes! The Turtle graphics window will display a beautiful spirograph pattern gradually evolving as the program runs. The colors will change gracefully, creating a visually stunning masterpiece.

Experiment and Customize

The beauty of this Python Turtle spirograph is that you can customize it to your heart's content. Modify parameters like tracer() to control the animation speed, adjust pensize() to change the thickness of the lines, or experiment with the h variable to create different color palettes.
